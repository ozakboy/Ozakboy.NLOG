## 支援架構

1. .net core 3.1
2. .net 6.0
3. .net 7.0
4. .NET Standard 2.0 2.1

## 功能簡介

建立 Log.txt 在專案擋案內，方便快速紀錄LOG 不寫入資料庫

自用、長期維護、緩慢更新

## 可以使用的方法

* Trace_Log    
* Debug_Log
* Warn_Log
* Error_Log
* Fatal_Log
* CostomName_Log
* SetLogKeepDay
* SetBigFilesByte
