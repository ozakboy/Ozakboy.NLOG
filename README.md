## 要求

ASP.net Core 6

## 功能

建立 Log.txt 在專案擋案內，方便快速紀錄LOG 不寫入資料庫

## 可以使用的方法

* Trace_Log    
* Debug_Log
* Warn_Log
* Error_Log
* Fatal_Log
* CostomName_Log
* SetLogKeepDay
* SetBigFilesByte
